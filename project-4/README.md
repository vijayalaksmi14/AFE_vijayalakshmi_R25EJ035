#  Tech & AI News Lounge

A clean, responsive, and modern news hub dashboard that aggregates recent updates across Artificial Intelligence, Web Development, and Open-Source ecosystems. Built with a structured, component-driven HTML layout and utility-focused CSS Grid styling.

---

## Key Features

* **Grid Architecture:** Uses dynamic CSS Grid templates (`repeat(auto-fit, minmax(300px, 1fr))`) to automatically realign news columns depending on screen dimensions.
* **Component Styling:** Modular card designs featuring isolated article text, specific categorical metadata labels, and visual shadow boundaries.
* **Modern Color Palette:** Built using soft background offsets (`#fafafa`) contrasting against sleek accent badges and dark navigations.

---

##  Built With

* **HTML5:** Semantic architecture including structured content divisions and standalone card layouts.
* **CSS3:** Responsive typography, CSS properties initialization (`:root`), Flexbox properties, and CSS Grid layouts.
